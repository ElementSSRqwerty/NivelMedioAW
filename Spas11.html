Software libre

Escritorio

7-zip

7-zip es un copmresor/descompresor de archivos que maneja los formatos ZIP, RAR, CAB, GZIP, BZIP2 y TAR, asi como un pofmato propio, 7z.

Audacity

Audacity es un editor de audio. Permite reproducir, grabar, editar, mezclar, aplicar efectos, etc...

GIMP

GIMP (GNU Image Manipulation Program) es un programa de retoque fotográfico y composición y edición de imágenes.

Inkacape

Inkscape es un editor de gráficos vecoriales que utiliza el formato SVG y que tambien importa formatos como Postscript, EPS, JPEG, PNG y TIFF y exporta en PNG y varios formatos vectoriales.

LibreOffice

LibreOffice es una suite ofimatica completa compuesta de procesador de textos, hojas de calculo, presentadores, programa de dibujo, base de datos y editor de ecuaciones.

Internet

Filezilla

Filezilla es un cliente de FTP.

Firefox

Firefox es un navegador web altamente personalizable y ampliable mediante los cientos de extensiones y temas disponibles.

HTTrack

HTTrack permite descargar sitios webs completos para su consulta offline.

Pidgin

Pidgin es un cliente de mensajería instantánea, compatible con las redes AIM (protocolos OScar y TOC), ICQ, MSN Messenger, Yahoo, IRC, Jabber, Gadu-Gadu, Zephyr y otras.

Thunderbird

Thunderbird es un cliente de correo de los creadores de Firefox. Incorpora multitud de caracteristicas avanzadas y, como Firefox, puede personalizarse mediante temas y ampliar sus posibilidades mediante extensiones.
